# 介绍

在Vue3中，@vue/reactivityopen in new window作为独立的package子包，可以脱离Vue在其他工具和库中进行使用，你甚至可以在React中使用。

之所以Vue3能够这样，而Vue2不行，这是因为Vue3采用Monorepo进行项目代码管理，它让各个模块之间，能够相互独立进行发包。

如果你对Monorepo还不是特别了解的话，你可以点击Monorepo + Rollupopen in new window这个章节去了解更多内容。

在这一章节，我们重点分析reactivity模块中各个API是如何实现的，包括：ref、reactive、computed以及readonly等等。
