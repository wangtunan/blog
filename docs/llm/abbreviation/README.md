# 名词缩写

## 基础概念(Core Concepts)
| 缩写      | 全称                              | 含义     |
| ------- | ------------------------------- | ------ |
| **AI**  | Artificial Intelligence         | 人工智能   |
| **ML**  | Machine Learning                | 机器学习   |
| **DL**  | Deep Learning                   | 深度学习   |
| **LLM** | Large Language Model            | 大语言模型  |
| **AGI** | Artificial General Intelligence | 通用人工智能 |
| **NLP** | Natural Language Processing     | 自然语言处理 |
| **CV**  | Computer Vision                 | 计算机视觉  |
| **ASR** | Automatic Speech Recognition    | 语音识别   |
| **TTS** | Text To Speech                  | 文本转语音  |
| **VLM** | Vision Language Model           | 视觉语言模型 |

## 模型结构相关(Model Architecture)
| 缩写              | 全称                           | 含义          |
| --------------- | ---------------------------- | ----------- |
| **NN**          | Neural Network               | 神经网络        |
| **DNN**         | Deep Neural Network          | 深度神经网络      |
| **CNN**         | Convolutional Neural Network | 卷积神经网络      |
| **RNN**         | Recurrent Neural Network     | 循环神经网络      |
| **LSTM**        | Long Short-Term Memory       | 长短期记忆网络     |
| **GRU**         | Gated Recurrent Unit         | 门控循环单元      |
| **Transformer** | —                            | 当前 LLM 主流架构 |
| **MoE**         | Mixture of Experts           | 专家混合模型      |
| **MLP**         | Multi-Layer Perceptron       | 多层感知机       |

## Attention相关
| 缩写           | 全称                           | 含义       |
| ------------ | ---------------------------- | -------- |
| **SA**       | Self Attention               | 自注意力     |
| **MHA**      | Multi Head Attention         | 多头注意力    |
| **KV Cache** | Key Value Cache              | 推理缓存     |
| **RoPE**     | Rotary Position Embedding    | 旋转位置编码   |

## 训练方法相关(Training)
| 缩写        | 全称                                         | 含义       |
| --------- | ------------------------------------------ | -------- |
| **PT**    | Pre-Training                               | 预训练      |
| **FT**    | Fine-Tuning                                | 微调       |
| **SFT**   | Supervised Fine-Tuning                     | 监督微调     |
| **RLHF**  | Reinforcement Learning from Human Feedback | 人类反馈强化学习 |
| **RLAIF** | Reinforcement Learning from AI Feedback    | AI反馈强化学习 |
| **DPO**   | Direct Preference Optimization             | 偏好优化     |
| **LoRA**  | Low Rank Adaptation                        | 低秩微调     |
| **QLoRA** | Quantized LoRA                             | 量化 LoRA  |

## RAG相关

| ------------- | ------------------------------ | ------ |

## 评测指标相关(Evaluation Metrics)
| 缩写                   | 全称                                                | 含义      |
| -------------------- | ------------------------------------------------- | ------- |
| **BLEU**             | Bilingual Evaluation Understudy                   | 翻译评估    |
| **ROUGE**            | Recall-Oriented Understudy for Gisting Evaluation | 摘要评估    |
| **Perplexity (PPL)** | 困惑度                                               | 语言模型质量  |
| **Accuracy**         | 准确率                                               | 分类任务    |
| **F1**               | F1 Score                                          | 精确率+召回率 |
